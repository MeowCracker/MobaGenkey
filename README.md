# MobaGenkey

你应该知道这是干啥的

## 开始使用

先安装环境

```shell
$ pip install -r requirements.txt
```

然后直接运行

```shell
$ python app.py
```

打开浏览器访问控制台展示的地址（例如[http://127.0.0.1:8000](http://127.0.0.1:8000)），按照网页的指示自己玩
